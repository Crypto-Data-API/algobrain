# Data Mining

In a trading-research context, "data mining" refers to the multiple-testing / selection-on-the-best bias that turns noise into apparent edge. See [[data-snooping-and-p-hacking]] for the canonical treatment — the multiple-comparisons problem, deflated Sharpe ratio, and the defenses (pre-registration, hold-out sets, walk-forward, honest N counting).
