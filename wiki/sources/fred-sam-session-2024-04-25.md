## Fred's Decision Framework (demonstrated)

This session reveals Fred's decision process in action:
1. Start with macro thesis (why gold?)
2. Check if a specific stock fits (Macquarie recommendation as prompt)
3. Run through fundamental metrics systematically
4. Identify red flags (PE too high, ROE too low)
5. Weigh positives (low D/E, good EBITDA, dividend payer) against negatives
6. Reach conclusion: "fully priced" = don't buy, but watch for dip
