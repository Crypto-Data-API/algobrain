# Best YouTube Channels for Quant and AI Trading

YouTube is an underrated resource for learning algorithmic and AI-driven trading. These channels provide free video tutorials ranging from beginner Python scripting to advanced ML model implementation. Video format is especially useful for coding walkthroughs where you can follow along in real time.

## Python and Quantitative Trading

**QuantPy** — Focused Python tutorials for quantitative finance. Covers portfolio optimization, options pricing, technical indicators in code, and statistical analysis. Clean, well-structured videos aimed at intermediate Python users. Best for: systematic application of Python to quant problems.

**Sentdex** — One of the largest Python tutorial channels. Extensive series on Python for finance covering data fetching, analysis, and basic ML. Also covers general ML and deep learning applicable to trading. Beginner-friendly with hundreds of videos. Best for: broad Python and ML foundation.

**AlgoTrading101** — Structured educational content on algorithmic trading concepts, strategy types, and platform walkthroughs. Less coding-heavy, more conceptual. Good starting point before diving into implementation. Best for: understanding algo trading concepts.

## Crypto and Bot Building

**Part Time Larry** — Focused on building crypto trading bots with Python. Covers exchange APIs (via CCXT), Freqtrade strategy development, data analysis, and live bot deployment. Practical and project-based. Best for: hands-on crypto bot development.

**The Trading Parrot** — Backtesting-focused content using Freqtrade and other frameworks. Strategy optimization, hyperparameter tuning, and realistic performance evaluation. Connects directly to [[backtesting-checklist]] principles. Best for: proper backtesting methodology.

## Machine Learning and Projects

**Coding Jesus** — Trading algorithm development with ML techniques. Builds complete projects from data collection through model training to strategy evaluation. Good pace for intermediate learners. Best for: end-to-end ML trading projects.

**Nicholas Renotte** — ML and deep learning project tutorials including finance applications. Covers TensorFlow and PyTorch implementations, model deployment, and practical tips. Best for: deep learning implementation patterns.

## How to Learn Effectively from YouTube

1. **Code along** — Pause the video and type every line. Passive watching teaches nothing
2. **Modify the examples** — Change parameters, try different assets, break things intentionally
3. **Watch at 1.5x speed** — Most tutorial pacing is slow; speed up and pause when needed
4. **Check the dates** — Older videos may use deprecated APIs or library versions
5. **Cross-reference** — Combine YouTube tutorials with documentation from [[python-quant-stack]] and discussions in [[trading-communities]]

## Skill Level Guide

- **Beginner**: Sentdex, AlgoTrading101
- **Intermediate**: QuantPy, Part Time Larry, The Trading Parrot
- **Advanced**: Coding Jesus, Nicholas Renotte (ML-focused projects)
