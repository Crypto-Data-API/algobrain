# Python for Finance — Udemy Courses

Udemy hosts several popular Python-for-finance courses that serve as the fastest on-ramp for beginners who want to combine programming with financial markets. The most notable instructors include Jose Portilla, Mayank Rasu, and Alexander Hagmann. These courses regularly go on sale for $10-20 and provide solid foundations.

## Key Courses

**Python for Financial Analysis and Algorithmic Trading (Jose Portilla)** — The most popular option. Covers pandas, NumPy, matplotlib, and basic backtesting with real stock data. Walks through connecting to APIs, calculating returns, and building simple moving average crossover strategies.

**Algorithmic Trading with Python (multiple instructors)** — More strategy-focused courses that cover momentum, mean reversion, and pairs trading. Some include API connections to brokers for paper trading.

**Python for Finance: Investment Fundamentals & Data Analytics** — Covers portfolio theory, CAPM, Monte Carlo simulation, and basic risk analysis using Python. Good bridge between finance theory and code.

## What You Will Learn

- Data manipulation with pandas — loading, cleaning, and analyzing price data
- Visualization with matplotlib and plotly for chart analysis
- Basic technical indicators and how to compute them in code
- Simple backtesting loops and strategy evaluation
- API connections to Yahoo Finance, Alpha Vantage, and broker APIs
- Portfolio metrics: Sharpe ratio, drawdown, returns analysis

## Strengths

- Extremely beginner-friendly — assumes no prior finance or coding knowledge
- Hands-on from the start with real market data
- Lifetime access once purchased
- Active Q&A sections where instructors respond
- Affordable ($10-20 during frequent Udemy sales — never pay full price)

## Limitations

These courses will not make you a quant. They cover the basics and stop well short of ML, advanced statistics, or production-grade systems. Think of them as stepping stones to [[ml-for-trading-specialization]] or [[quantconnect-bootcamp]].

## Who Should Take These

Complete beginners who want to go from zero Python to running their first backtest in a weekend. If you already know Python and pandas, skip directly to [[ml-for-trading-specialization]] or [[fast-ai-for-trading]].

## Tips

Wait for a Udemy sale — they happen almost every week. Never pay more than $20. Use the [[python-quant-stack]] page as your reference for which libraries to install alongside the course material.
