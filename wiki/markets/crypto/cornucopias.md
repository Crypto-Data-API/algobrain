# Cornucopias

**Cornucopias** (COPI) is a cryptocurrency. It ranks **#2259** by market capitalization. 'The Island' is a massive Play-To-Earn, Build-To-Earn, and Learn-To-Earn blockchain based game where players can be rewarded with and/or own land, properties and other NFT based assets with real world value all by playing games in a fun and safe metaverse.
