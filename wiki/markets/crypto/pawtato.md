## Overview

Pawtato Finance is an all-in-one DeFi and GameFi ecosystem built on the Sui blockchain. It provides users with practical tools for wallet management, real-time notifications, portfolio tracking, and on-chain actions through a unified Web and Telegram platform. The project also includes Pawtato Land, an on-chain strategy game where users own assets, generate resources, and progress through blockchain-verified gameplay. The $TATO token connects all products and is used for utility, governance, and in-game functions. Pawtato Finance focuses on delivering functional tools, transparent mechanics, and a streamlined user experience across DeFi, NFTs, and gaming on Sui.
