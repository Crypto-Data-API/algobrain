# HUNDRED

> **Subject-integrity / disambiguation note (added 2026-06-22):** The verified CoinGecko data for this page (ticker **HUNDRED**, website **hundredcoin.io**, native chain **Avalanche**, category **Meme**, contract `0x4586...`) describes an **Avalanche memecoin**. This is **NOT** *Hundred Finance* (ticker **HND**), the cross-chain Compound-fork [[lending]] protocol that suffered an ~$7M reentrancy/[[donation-attacks|donation]] exploit in April 2023 (documented in [[compound-fork-donation-short]] and [[donation-attacks]]). The two share part of a name but are unrelated assets. This page has therefore **not** been upgraded with Hundred Finance content, to avoid misattribution. Only the market-data figures have been refreshed. See log/report.

**HUNDRED** is a self-described Avalanche memecoin (the project's own copy markets an "anti-paper-hands" 100-hour transfer lock). It ranks **#812** by market capitalization. The text below is reproduced from the project's own promotional material and is not an endorsement or a factual claim by this wiki.
