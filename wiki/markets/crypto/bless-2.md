## Overview

Bless Network is a decentralized computing platform designed to transform idle capacity from everyday devices into a global shared computer. Founded in SanFrancisco, USA, under the name Blockless in 2022.
By leveraging laptops, desktops, and mobile devices, Bless enables applications to access distributed computing resources directly from their user base. As moreusers join, the available network capacity scales proportionally, creating a self-sustaining infrastructure layer.
The network is built on its neutral application framework (nnApp) and nested node infrastructure, which allow developers to deploy compute-intensive applicationsacross heterogeneous devices. This model reduces dependence on centralized cloud providers, a common bottleneck for decentralized ecosystems.
