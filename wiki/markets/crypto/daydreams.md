## Overview

Daydreams is a generative agent library for playing anything onchain. It is chain agnostic and can be used to perform onchain tasks - including play any onchain game - by simply injecting context. 

Daydreams is MIT Licensed, and built by developers with deep expertise in gaming and blockchain infrastructure. The library is chain agnostic, agent framework agnostic and game agnostic. You will be able to plug and play any game or agent framework into Daydreams.

It is designed to be as lite as possible. Keep it simple and powerful.
