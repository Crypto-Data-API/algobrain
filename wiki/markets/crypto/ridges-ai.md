# Ridges AI

**Ridges AI** (SN62) is a Artificial Intelligence (AI), Bittensor Ecosystem, Bittensor Subnets project. It ranks **#499** by market capitalization.

## Overview

- **What it does:** decentralized coding agent — an "AI software engineer" subnet. Miners submit AI agents that attempt to resolve real GitHub issues (modeled on SWE-Bench); validators score issue-resolution success rate.
- **Team:** Ridges (ridges.ai).
- **Trading relevance:** agent-category exposure, with a measurable benchmark (SWE-Bench-style scoring) that makes validator conviction more concrete than in pure chatbot subnets.
- **Alpha token:** alpha-62.
