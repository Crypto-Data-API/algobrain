# Base (Coinbase L2)

**Base** is an [[ethereum|Ethereum]] [[layer-2]] rollup built by [[coinbase|Coinbase]] using the OP Stack (Optimism's open-source rollup framework). Launched in August 2023, Base is designed to be a low-cost, developer-friendly chain that leverages Coinbase's 100M+ verified user base for onboarding. It has no native token — transaction fees are paid in ETH.

## Why Base Matters

Base's significance comes from **distribution, not technology**. As Coinbase's native L2:

- **Direct onboarding**: Coinbase can funnel 100M+ users directly to Base with one-click bridging
- **USDC native**: [[usdc|USDC]] is natively deployed on Base via [[cctp|CCTP]], making it a first-class stablecoin chain
- **Institutional credibility**: Coinbase is a US-regulated, publicly traded company (NASDAQ: COIN)
- **Developer ecosystem**: Strong dev tooling and grants program attracted rapid ecosystem growth

## USDC on Base

Base is a critical chain for [[usdc|USDC]] adoption:

- Native USDC via [[cctp|CCTP]] — no wrapped tokens
- Coinbase offers free USDC transfers from the exchange to Base
- Sub-penny transaction costs make USDC micropayments viable
- Coinbase Commerce integration enables merchant payments in USDC on Base

This makes Base one of the most practical chains for real-world stablecoin usage.

## Relationship to Optimism Superchain

Base contributes a portion of its sequencer revenue to the Optimism Collective as part of the Superchain arrangement. This aligns Base with Optimism's modular rollup vision while giving Coinbase operational control over its chain.

## Trading Relevance

- **No native token** means no direct "Base token" to trade — exposure comes through ecosystem tokens (Aerodrome's AERO) or through Coinbase stock (COIN)
- Base DEX volume and TVL metrics serve as indicators for Coinbase's on-chain strategy success
- [[usdc]] flows to Base (via [[cctp]]) indicate institutional and retail adoption trends
- Growing Base ecosystem competes with [[arbitrum|Arbitrum]], [[solana|Solana]], and other chains for DeFi market share
