# AIOZ Network

**AIOZ Network** (AIOZ) is a [[depin|DePIN]] (Decentralized Physical Infrastructure) project for **Web3 AI, storage and media streaming**, running its own EVM-compatible Layer 1 (with deployments also on [[ethereum|Ethereum]] and BNB Chain). A global network of node operators is rewarded in AIOZ for contributing bandwidth, storage and GPU compute used to deliver media (decentralized CDN/streaming) and power decentralized AI inference.

## Decentralized Infrastructure: CDN, Streaming, and Compute

AIOZ operates a **three-pillar decentralized infrastructure** model, each powered by a distributed network of node operators:

### 1. Decentralized Content Delivery Network (dCDN)

AIOZ's original product is a decentralized CDN that distributes media content through a peer-to-peer network of nodes rather than centralized servers (like Cloudflare or Akamai).

- **How it works:** Node operators run AIOZ software on their computers, contributing bandwidth and storage. When a user requests content, it is delivered from the nearest available node
- **Advantage:** Lower cost than centralized CDNs; censorship-resistant; geographically distributed
- **Revenue model:** Content publishers pay AIOZ tokens for CDN services; node operators earn AIOZ for serving content

### 2. Decentralized Streaming

Built on top of the dCDN, AIOZ supports decentralized video streaming -- a Web3 alternative to YouTube/Twitch content delivery infrastructure.

- **Use case:** Video platforms that want to reduce CDN costs or operate outside centralized infrastructure
- **Competitive landscape:** Livepeer (video transcoding), Theta Network (video delivery), and traditional CDNs are competitors

### 3. Decentralized AI Compute

AIOZ expanded into AI compute, allowing node operators to contribute GPU resources for AI inference and training workloads. This puts AIOZ in competition with [[akash-network|Akash]], [[render-token|Render]], and [[aethir]] in the decentralized compute market.

## Web3 Infrastructure Play

AIOZ positions itself as a **full-stack decentralized infrastructure** provider -- not just compute or just CDN, but an integrated layer covering data delivery, storage, and computation. This "infrastructure convergence" thesis argues that decentralized networks that provide multiple services can achieve better utilization and node economics than single-purpose networks.

### Node Economics

| Metric | Detail |
|---|---|
| **Node types** | Edge nodes (CDN), compute nodes (AI), storage nodes |
| **Rewards** | AIOZ tokens for bandwidth served, compute provided, and storage contributed |
| **Requirements** | Consumer hardware sufficient for CDN nodes; GPU required for compute nodes |
| **Staking** | Nodes may stake AIOZ as collateral to participate |

### Layer 1 Blockchain

AIOZ runs its own **Layer 1 blockchain** (EVM-compatible) for settlement, governance, and node coordination. This is notable because most DePIN projects build on existing chains (Ethereum, Solana, Cosmos). Running an L1 gives AIOZ sovereignty but adds the challenge of bootstrapping validator security and ecosystem.

## Valuation Framing (qualitative)

AIOZ is a **small-cap DePIN infrastructure token** whose valuation hinges on demonstrable usage, which is currently hard to verify:

- **Usage opacity** — unlike [[aethir]] (which publishes container counts and ARR) or [[akash-network|Akash]] (lease/revenue metrics), AIOZ does not transparently report bandwidth served, storage filled, or compute hours sold. Without these, the token is valued on narrative, not fundamentals.
- **Supply quality** — the ~0.99 MC/FDV is a genuine positive: re-rating would not be diluted away by scheduled unlocks (only by uncapped emissions).
- **Sector beta** — AIOZ is a high-beta proxy on the DePIN/AI basket; it tends to rally on NVIDIA/AI catalysts and sell off when the AI-compute premium compresses.
- **Convergence bet** — the bull case is "infrastructure convergence" (CDN + streaming + storage + AI compute on one network beats single-purpose rivals); the bear case is that this spreads a small team too thin against specialized competitors.

This is a qualitative framing, not a price target.
