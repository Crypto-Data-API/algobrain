# Axelar

**Axelar** (ticker: **AXL**) is a [[cosmos|Cosmos]]-SDK proof-of-stake blockchain built specifically as a [[cross-chain|cross-chain]] [[interoperability|interoperability]] layer, letting developers and dApp users move assets and call smart contracts across any connected chain with a single API. It is often described as "Stripe for Web3": validators run light clients and perform multiparty signing to route and secure messages between [[ethereum|Ethereum]], EVM L2s, Cosmos chains, and others. The AXL token is used for governance, staking, and paying network transaction fees. It sits in the cross-chain [[bridge|bridge]] / interoperability category alongside [[wormhole|Wormhole]].

## Overview

Axelar delivers secure interchain communication. That means dApp users can interact with any asset, any application, on any chain, with one click. You can think of it as Stripe for Web3. Developers interact with a simple API atop a permissionless network that routes messages and ensures network security via proof-of-stake consensus.

Axelar is a blockchain that connects blockchains. To do this securely, Axelar network uses proof-of-stake consensus. Network validators produce new blocks, participate in multiparty signing and vote on external chain states. Tokenholders stake the AXL token, delegating tokens to a validator’s staking pool and receiving rewards, minus the validator’s commission.

The AXL token is used for governance and to pay network transaction fees to validators and tokenholders. However, users of the Axelar network do not need to hold the token. Conversions into AXL tokens and destination-chain tokens are handled on the back end, so that users need only pay gas once, in the source-chain token.

Axelar network rewards are “inflationary” — they increase the total supply of AXL — but there is potential for supply to become deflationary based on the transaction fee mechanic described above.

## Cross-Chain Messaging & Bridge Security

Axelar's core product is **General Message Passing (GMP)** — a contract on a source chain can trigger logic on a destination chain — rather than just token transfers. The security model differs structurally from a federated multisig or off-chain relayer:

- **Standalone PoS validator set:** Axelar is its own [[cosmos|Cosmos]]-SDK / Tendermint blockchain whose only job is to validate and route [[cross-chain|cross-chain]] messages. Validators run **light clients** of connected chains, observe events, and perform **threshold (multiparty) signing** to attest state — replacing a single multisig key with a stake-weighted committee.
- **Economic security:** the cost of corrupting the network scales with the value of AXL staked. A low token price shrinks that security budget, a recurring concern for all bridge/validator-secured [[interoperability|interoperability]] networks.
- **Key infrastructure:** Satellite (bridge UI), Axelar Gas Service (users pay gas once in the source token), and Interchain Token Service (ITS) for canonical multichain tokens.

This makes Axelar a "blockchain that connects blockchains" — its differentiator versus [[wormhole|Wormhole]] (Guardian attestations) and LayerZero (oracle + relayer) is that message verification is performed by an on-chain, slashable PoS consensus rather than an off-chain set. As with all bridges, however, value concentrates behind a shared trust assumption — see [[cross-chain-bridge-risk]] and [[bridge]].

## Valuation Framing (qualitative)

AXL is an **infrastructure / interoperability bet** whose token value tracks cross-chain message volume, fee capture, and the security budget required to keep the validator set honest. Bull case: a multi-chain world routes growing value through neutral plumbing, GMP/ITS adoption rises, and fee burn turns net issuance deflationary. Bear case: interoperability is crowded ([[wormhole|Wormhole]], LayerZero, CCIP, native IBC) and commoditizing, uncapped inflation dilutes holders in a low-volume regime, and a low token price weakens economic security. As a sub-$100M infra token near its all-time low in an Established Bear Market, AXL carries high regime and competitive sensitivity. *This is framing, not a price target or recommendation.*

## Use Case, Narrative & Category

Axelar belongs to the **cross-chain interoperability / bridge** narrative alongside [[wormhole|Wormhole]], LayerZero, and Chainlink CCIP. Its differentiator is being a **standalone PoS blockchain** (built on the [[cosmos|Cosmos]] SDK and Tendermint consensus) whose only job is to validate and route cross-chain messages — rather than a federated multisig or off-chain relayer set. Validators run light clients of connected chains, observe events, perform threshold/multiparty signing, and produce General Message Passing (GMP) calls that let a contract on one chain trigger logic on another.

Key products include the Satellite bridge UI, Axelar Gas Service (so users pay gas once in the source token), and Interchain Token Service (ITS) for deploying canonical multichain tokens. The investment thesis is that secure, programmable interoperability is foundational infrastructure for a multi-chain world.

## Tokenomics Note

AXL has an **uncapped (inflationary) max supply** — staking rewards mint new AXL — but the design includes a fee-burn mechanic that can make net issuance deflationary if cross-chain transaction volume is high enough. The circulating-to-total ratio is high (market cap ≈ FDV), so near-term unlock-driven dilution is limited compared with many newer tokens.

## Risks

- **Bridge / interoperability risk:** Although Axelar replaces a single multisig with a PoS validator set, cross-chain protocols remain prime targets — a consensus or signing failure could compromise bridged value. See [[cross-chain-bridge-risk]].
- **Validator security budget:** Security scales with the value staked in AXL; a low token price can shrink the economic cost of attacking the network.
- **Inflation:** Uncapped supply means holders are diluted unless fee burn offsets issuance.
- **Competitive pressure:** Interoperability is crowded (Wormhole, LayerZero, CCIP, native IBC), pressuring fees and adoption.
- **Macro / liquidity risk:** As a sub-$100M-cap altcoin, AXL is highly sensitive to the broad risk-off regime (Fear & Greed 23 / Established Bear Market as of 2026-06-20).
