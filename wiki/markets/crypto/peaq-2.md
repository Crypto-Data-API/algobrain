# peaq

**peaq** (PEAQ) is a Layer 1 blockchain purpose-built for [[depin|DePIN]] — the economic and coordination layer for "machines doing business." It gives robots, vehicles, devices, sensors, and AI agents the primitives to become autonomous economic actors: verifiable machine identities (Self-Sovereign Machine Identity), machine wallets, portable reputation, tokenization, and cross-chain settlement. Originally part of the [[polkadot|Polkadot]] ecosystem, peaq runs its own L1 and is increasingly positioned as a chain-neutral, omnichain coordination layer for machine economies ("Robot Money").

PEAQ is the native gas, staking, and governance token of the network. peaq has become a hub host for numerous DePIN applications (mobility, energy, connectivity) that deploy on or coordinate through its chain.

## Technology & Consensus

peaq is a [[layer-1]] blockchain originally built with the **Substrate** framework (the [[polkadot|Polkadot]] SDK) and historically run as a Polkadot parachain, now operating its own L1 with an increasingly chain-neutral, omnichain posture. Its consensus is a [[proof-of-stake]] / nominated-staking model where validators secure the chain and PEAQ is the staked asset. The differentiating stack is purpose-built for [[depin|DePIN]] and the "machine economy":

- **Self-Sovereign Machine Identity (SSMI)** — verifiable, on-chain identities for robots, vehicles, sensors, and AI agents.
- **Machine wallets & Machine DeFi** — devices and agents can hold balances, transact, and access financial primitives autonomously.
- **peaq functions** — modular pallets for DePIN builders (identity, access control, payment, role-based permissioning) so hardware networks can launch without rebuilding core machine-economy logic.
- **Cross-chain settlement** — omnichain orientation so machine transactions can settle across ecosystems rather than locking DePIN apps to one chain.

peaq is the **base layer many DePIN hardware networks build on** rather than a single device fleet, so value accrues from aggregate machine/DePIN activity. It also ties into the **x402** agent-payments ecosystem and the broader robotics / AI-agent narrative.

## Use Case, Narrative & Category

peaq is a flagship **DePIN Layer 1**, also tagged under Smart Contract Platform, Layer 1, Polkadot Ecosystem, Robotics, and the x402 (agent-payments) ecosystem. It targets the "machine economy" / DePIN coordination thesis: as physical infrastructure networks proliferate (mobility, energy, sensing, robotics), they need a neutral settlement and identity layer where machines can hold wallets, build reputation, and transact across chains.

The hardware-network economics angle: peaq itself is not a single hardware network but the **base layer that many DePIN hardware networks build on**, so its value accrues from aggregate machine/DePIN activity rather than one device fleet. peaq's "Machine / Trust / Service" layering and emphasis on Robot Money and AI-agent payments place it at the intersection of DePIN, robotics, and the agentic-payments narrative. The bear case: DePIN-L1 demand is still early, peaq competes for DePIN mindshare with [[solana]] and other general-purpose chains, and a dedicated L1 must bootstrap both security and app demand simultaneously.

## Overview

peaq is the economic system enabling robots and machines to do business on every chain. It provides a neutral, omnichain coordination layer that gives machines, including robots, vehicles, devices, sensors, and AI agents, everything they need to become autonomous economic actors. This includes verifiable identities, wallets, portable reputation, tokenization, and cross-chain settlement guarantees. 

peaq is structured in three layers. The Machine Layer (peaqOS) onboards any machine as a composable economic actor. The Trust Layer provides coordination, validation, and economically secured reputation scores backed by staked capital. The Service Layer connects machines to external service markets through a permissionless adapter framework for discovery, scoring, and orchestration. 

peaq is chain-neutral by design, operating as the shared trust and coordination layer for machines across existing blockchains. A machine onboarded through peaq exists across all supported chains simultaneously, and any application on any chain can discover, verify, and transact with it. peaq enables Robot Money, any medium of exchange, measure of value, or means of payment that robots and machines use across any chain.
