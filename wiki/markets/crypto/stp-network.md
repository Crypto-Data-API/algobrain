# AWE Network (formerly STP Network / STPT)

**AWE Network** (ticker **AWE**) is the rebrand of the long-running **STP Network** (ticker **STPT**) — the CoinGecko entry under id `stp-network` now reports symbol **AWE** and name **AWE Network**. The project has pivoted from its earlier "Standard Tokenization Protocol" identity to an **AI "Autonomous Worlds"** thesis on [[base|Base]]: the Autonomous Worlds Engine (AWE) is a modular framework for self-sustaining worlds in which AI agents collaborate, adapt, and evolve, with agent-agent and human-agent interaction at scale. AWE is the network's native utility/governance token.

> **Identity note:** "stp-network" is the legacy slug. STPT was an older-generation token (ATH dates back to 2021, ATL to 2019). The current asset trades as AWE following the rebrand/migration; holders of legacy STPT should confirm any swap/migration mechanics from official channels.

## Architecture / how it works

The **Autonomous Worlds Engine (AWE)** is pitched as infrastructure for AI-agent ecosystems:

- **Modular world framework** — tooling to spin up self-sustaining "autonomous worlds" populated by AI agents that can perceive, act, and persist state over time.
- **Multi-agent coordination at scale** — claims to coordinate thousands of agents via parallel processing, dependency management, and GPU-optimised workloads, so agents can collaborate rather than run as isolated bots.
- **World.Fun** — an "autonomous worlds launcher" supporting AI-driven worlds (advertised up to ~1,000 agents per world) powered by AWE; users create/customise agents for deployment, lowering the barrier to standing up an agent world.
- **Base-native settlement** — the token and core contracts live on [[base|Base]], aligning AWE with the Coinbase L2 ecosystem and its growing consumer/AI app footprint.

These are project claims and are **not independently verified** within this wiki. The pivot from STP Network's tokenization-protocol roots to an AI-agents platform is a substantial change of thesis; weigh execution risk accordingly. Distinguish the verified facts (mature token, Base contract, broad exchange listings, near-full circulation) from the self-reported product roadmap.

## Value accrual / governance

AWE functions as the **utility and governance** asset of the Autonomous Worlds ecosystem: it is intended to pay for world creation/agent deployment via World.Fun, to coordinate the agent economy, and to govern the protocol. Because the token is essentially fully circulating, the value question is cleanly about **usage demand** — how many worlds/agents are actually deployed and how much AWE that consumes — rather than about future unlock absorption. The risk is that, like many AI-narrative tokens, demand is currently more reflexive (sentiment-driven) than fundamentally usage-driven.
