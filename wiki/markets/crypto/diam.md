## Overview

Diamante is a Layer 1 network focused on quantum-proof security for everyday crypto use. It combines a modern proof-of-stake validator set with an execution layer designed for low latency finality and high throughput.

The network uses post-quantum signature schemes alongside standard cryptography, plus an execution model built to process transactions in parallel where possible. Developers can deploy apps through a familiar toolchain, and users can move assets, run privacy-preserving connections, and interact with on-chain services.

Diamante prioritizes quantum-proof primitives for keys and signatures, and integrates AI-assisted telemetry to optimize routing and performance across nodes. The design aims to keep consumer apps responsive while maintaining strong security assumptions.

DIAM pays network fees, is staked by validators and delegators to secure the chain, and participates in on-chain governance for protocol upgrades. Supply, emissions, and validator details are documented in the project docs and explorer.

Diamante is developed by Diamante with contributions from an open developer community. The network and code are open source and maintained in public repositories.
