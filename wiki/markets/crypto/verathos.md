## Overview

Verathos delivers cryptographically verified AI on Bittensor Subnet 96. Every inference generates a cryptographic proof over committed model weights, and validators verify random layers on CPU in approximately 4 milliseconds — cheap enough that any machine can validate, with no specialized hardware required. Proof failure results in instant probation, a zeroed score, and exclusion from traffic, eliminating the need to trust any individual operator. The product is an OpenAI-compatible API designed as a drop-in replacement for existing integrations. Verified chat is live at verathos.ai/chat, and the public API is currently in preview. Verified inference is in production today; verified training is on the roadmap. TEE-based encrypted inference (Intel TDX, AMD SEV-SNP, NVIDIA Confidential Compute) is documented and forthcoming on mainnet.
