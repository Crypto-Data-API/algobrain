# Emerald Security Token

**Emerald Security Token** (EMRL.D) is a cryptocurrency. It ranks **#647** by market capitalization. EMRL.D tokenizes emerald assets through a dual-entity model — a Special Purpose Vehicle (SPV) backed by a $90M SKR certificate, paired with EMRL.D S.A.S., a Colombian subsidiary that conducts emerald sourcing and procurement operations. This structure combines real-asset backing with operational activities from the source.

A new model in alternative investments:
Real assets.
