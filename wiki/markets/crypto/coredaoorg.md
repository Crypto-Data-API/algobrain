# Core

**Core** (ticker **CORE**) is an EVM-compatible [[layer-1]] blockchain that aims to extend [[bitcoin]]'s security and capital to a programmable smart-contract environment. It is secured by **Satoshi Plus**, a hybrid consensus that combines Bitcoin's [[proof-of-work]] hashrate (via delegated/merged-style mining), Delegated [[proof-of-stake]] from CORE holders, and — distinctively — non-custodial Bitcoin staking. CORE is the network's gas, staking and governance token.

CORE has a fixed max supply of 2.1 billion — deliberately echoing [[bitcoin]]'s 21M cap — with roughly 59% in circulation. Emissions are released over ~81 years, allocated across node mining rewards, users/relayers, contributors, the treasury and the reserve. The token is staked (and used to delegate to validators) under the Satoshi Plus model and pays gas. A portion of supply secures the chain and earns yield alongside delegated Bitcoin and hashrate; CORE has also implemented fee-burning mechanics in some periods, which can offset emissions.

## Technology / Consensus

Core's **Satoshi Plus** consensus elects validators using three inputs: delegated Bitcoin [[proof-of-work]] hashrate, Delegated [[proof-of-stake]] of CORE tokens, and non-custodial **Bitcoin staking** (BTC holders lock BTC via time-locks to support Core validators and earn yield without giving up custody). The stated goal is to inherit Bitcoin-grade security while providing a Turing-complete, EVM-compatible execution layer. Core is fully EVM-compatible, so Ethereum-tooling and Solidity contracts port directly, and it markets itself as part of the broader "BTCfi" (Bitcoin DeFi) movement.

---

## Use Case / Narrative / Category

Core's narrative is **"BTCfi" — making idle Bitcoin productive** by letting BTC holders stake to secure an EVM smart-contract chain and earn yield, while extending Bitcoin's economic security to DeFi. Category tags include Smart Contract Platform, [[layer-1]], Bitcoin Sidechains, and SideChain. It competes within the fast-growing Bitcoin-L2/sidechain landscape.

---

## Valuation Framing

CORE's **MC/FDV is ~0.59** ($33.4M circulating vs $56.7M fully diluted) against a fixed 2.1B max supply — the [[bitcoin]]-echoing cap is a deliberate branding choice. The long ~81-year emission schedule means issuance is slow but persistent, while fee-burning in active periods can partially offset it. The core valuation question for "BTCfi" plays is whether non-custodial [[bitcoin]] staking actually attracts durable BTC TVL and generates fee revenue that accrues to CORE, versus competing Bitcoin-security designs ([[bitcoin]] L2s, Babylon, Stacks) splitting the same idle-BTC demand pool. Trading near all-time lows, CORE is priced for the skeptical case; it is a high-beta wager on the BTCfi narrative converting into real on-chain economic activity.

## Overview

Powered by a new consensus mechanism, Satoshi Plus, Core is a Turing-complete blockchain leveraging the Bitcoin mining hashrate and the Ethereum Virtual Machine (EVM). Satoshi Plus applies a protocol-driven validator election mechanism to combine the optimal features of Proof of Work (PoW) and Delegated Proof of Stake (DPoS) in order to ensure the maximization of security, scalability, and decentralization.
