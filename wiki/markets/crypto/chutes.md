# Chutes

**Chutes** (SN64) is a Artificial Intelligence (AI), Bittensor Ecosystem, Bittensor Subnets project. It ranks **#333** by market capitalization.

## Overview

- **What it does:** serverless GPU inference — users deploy AI models as "chutes" (containerized inference endpoints) and the subnet routes inference requests across a decentralized set of miner GPUs. Competes with centralized serverless-GPU providers (Runpod, Replicate, Modal).
- **Team:** Chutes / Corcel / Rayon Labs cluster.
- **Trading relevance:** the flagship "Bittensor subnet with real revenue" story. Chutes has visible off-chain API revenue, which translates into reliable emission-share capture and makes alpha-64 one of the most liquid and most-held alphas. Frequently cited as the best single-subnet "fundamentals" bet.
- **Alpha token:** alpha-64.
