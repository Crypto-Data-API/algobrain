# ALEO

**ALEO** (ticker **ALEO**) is the native token of Aleo, a privacy-focused [[layer-1]] blockchain that uses [[zero-knowledge-proofs|zero-knowledge cryptography]] to enable fully private, programmable applications. Aleo is private-by-default: application logic runs off-chain and only succinct zero-knowledge proofs are posted on-chain, so user data stays confidential while remaining verifiable. ALEO is used for staking, transaction fees, and rewarding the network's proof-generation work.

## Technology / Consensus

Aleo is a [[layer-1]] built around [[zero-knowledge-proofs]]. Developers write applications in Aleo's own languages (Leo / Aleo instructions); execution happens off-chain and produces a zk-SNARK proof that is verified on-chain, giving "unbounded compute with privacy." Consensus combines a proof-of-stake validator set with a unique "Proof of Succinct Work" role for provers, who are economically incentivized to generate the zero-knowledge proofs that secure and scale the network. The design is private-by-default and open-source.

> Framing aid only, not a price target. A privacy L1's value hinges on application adoption that remains unproven at scale.

## Overview

Aleo was born from a desire to achieve the impossible - to build a user experience on the web that is both truly personal and truly private.

Aleo is the first platform to offer fully private applications.

Aleo achieves this by leveraging decentralized systems and zero-knowledge cryptography to protect user data on the web. At its core, Aleo offers users and application developers unbounded compute with absolute privacy.

By architecting Aleo as a blockchain that is private-by-default, open-source, and built for the web, we believe Aleo is uniquely positioned to address the shortcomings of blockchain adoption. With Aleo, users have access to a world of truly personalized web services without giving up control of their private data. 

Aleo will reshape the role decentralized systems play in our society and introduce new experiences never before seen on the web.
