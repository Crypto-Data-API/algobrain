# BitAgent (SN20)

**BitAgent** (SN20) is a Bittensor subnet for autonomous AI agents -- miners submit agents that perform tasks (tool use, API orchestration, multi-step reasoning); validators score task completion and efficiency. Part of the broader "AI agents on-chain" thesis.

## How It Works

The subnet runs task benchmarks against miner agents. Tasks include function-calling, tool-use chains, information retrieval with synthesis, and structured output generation. Validators score success rate, correctness, and cost (tokens consumed).

## Trading Relevance

Agent-category exposure on Bittensor, parallel to off-chain agent frameworks (LangChain, AutoGen, CrewAI). Alpha-20 performance correlates with broader agent-narrative sentiment. Closer to a sector proxy than a directly-tradable signal.
