## Overview

About HandlPay:
HandlPay is a peer-to-peer payments platform that turns social media usernames and WhatsApp numbers into private payment handles. It allows users to send and receive stablecoins using familiar identifiers, without sharing wallet addresses, exposing transaction history, or dealing with cross-chain complexity.
The experience is simple: if you can message someone, you can pay them.

Mission:
HandlPay’s mission is to make global payments as seamless as messaging, removing the technical barriers of crypto and enabling instant, private, and borderless transactions using only a social handle.

Supported Social Handles:
(Users can link and receive payments via)
- Gmail
- Instagram
- WhatsApp
- Telegram
- X

Supported Payment Methods:
- $USDC
- Credit &amp; debit cards
- Apple Pay

Supported Blockchains:
- Canton Network
- Ethereum
- Base
- Polygon
- Avalanche
- Binance Smart Chain

$HANDL Utility:
$HANDL is the core utility token powering the HandlPay ecosystem.

It is used to:
- Pay transaction fees within the platform
- Unlock premium features and enhanced functionality
- Incentivize user adoption and network growth

$HANDL is trading on MEXC and multiple DEXs across Base, Polygon, and Canton Network. For the most up-to-date list of trading pairs refer to the official token page on the HandlPay website.
