# three.ws

**three.ws** (THREE) is a platform that enables AI agents to exist as fully rendered 3D avatars that can be embedded across the web. It allows integration with different language models, so agents can be interactive, responsive, and used across multiple applications instead of being limited to simple chat interfaces.

The project is a blockchain-based token that directs a portion of its fees to support the developer @nichxbt, who has been onboarded. It ranks **#2093** by market capitalization.

## Overview

three.ws is a platform that enables AI agents to exist as fully rendered 3D avatars that can be embedded across the web. It allows integration with different language models, so agents can be interactive, responsive, and used across multiple applications instead of being limited to simple chat interfaces.

The project is a blockchain-based token that directs a portion of its fees to support the developer @nichxbt, who has been onboarded. It focuses on supporting the development of open, onchain AI infrastructure, including agent ownership, portability, and the ability for AI agents to operate across different platforms and ecosystems.
