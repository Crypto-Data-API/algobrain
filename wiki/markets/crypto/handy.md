# Handy

**Handy** (HANDY) is a blockchain-based game platform that provides virtual asset-related information and rewards utilizing virtual asset-based contents to users, and user pool from various nations to advertisers.
HANDY’s ‘HandyPick’ is a blockchain-based prediction game service that users predict the virtual asset prices and get rewards by the result of it.
HANDY will develop more blockchain-based games. It ranks **#610** by market capitalization.
