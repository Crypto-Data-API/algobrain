## Overview

UnityWallet is a self-custodial, multi-chain digital wallet designed to allow users to manage cryptocurrency and interact with decentralized services in a single mobile interface. The platform supports multi-chain asset management, token swaps, decentralized application access, and reward mechanics tied to user engagement. The UnityWallet Token (UNT) is the native utility token within the ecosystem. UNT is intended as a utility-oriented token that powers rewards, cashback, loyalty tiers, and related incentive mechanics.
