## Overview

Griffain is an **AI-agent automation platform** built on [[solana]]. Its product is an "Agent Engine" that lets users spin up autonomous agents which interpret natural-language commands and execute real on-chain transactions — swaps, DCA, NFT operations, and DeFi interactions — leveraging Solana's high throughput and low fees. The GRIFFAIN token is the unit of access/speculation around the platform and the focal point of the [[defai|DeFAI]] narrative.

> **Correction:** Some generic third-party listings describe Griffain as a "decentralized exchange / liquidity protocol." That is inaccurate. Griffain does **not** operate its own DEX or liquidity pools; its agents *route* through existing Solana venues like [[jupiter-exchange-solana|Jupiter]], [[raydium]], and [[orca]]. The earlier DEX framing has been corrected here; the agent-platform description is the verified one (Verified via WebSearch, 2026-06-11).

## AI Agent on Solana

Griffain is an **AI agent platform** built on [[solana]] that enables users to create and deploy autonomous AI agents capable of performing on-chain actions. Unlike [[aixbt]], which is primarily an analysis/alpha agent, Griffain's agents are designed to **execute transactions** -- swapping tokens, managing DeFi positions, and interacting with Solana protocols.

### What Griffain Agents Do

1. **Natural language commands:** Users instruct agents in plain English (e.g., "buy 100 USDC worth of SOL when price drops below $20")
2. **On-chain execution:** Agents execute actual Solana transactions through [[jupiter-exchange-solana|Jupiter]], [[raydium]], and other protocols
3. **Portfolio management:** Agents can monitor positions, rebalance allocations, and execute DCA strategies
4. **Social/memecoin trading:** Agents can be configured to snipe new token launches or trade based on social signals

### Agent Capabilities

| Capability | Description |
|---|---|
| **Token swaps** | Execute trades on Solana DEXes via Jupiter aggregation |
| **DCA execution** | Automated dollar-cost averaging into specified tokens |
| **Airdrop farming** | Agents interact with protocols to accumulate potential airdrop eligibility |
| **NFT operations** | Mint, list, and trade NFTs on Solana |
| **Custom strategies** | Users define conditional logic for agent behavior |

## Valuation Framing (Qualitative)

GRIFFAIN cannot be valued on cash flows — it has no protocol fee accrual to the token and trades as a **narrative/attention asset**. Useful qualitative anchors:

- **Full circulation, FDV = MC.** With ~100% of max supply circulating, there is no overhang of future unlocks diluting holders — a structural advantage versus VC-backed AI tokens still on vesting cliffs. The risk is entirely demand-side (attention), not supply-side.
- **Attention beta.** Price tracks the AI-agent / [[defai|DeFAI]] meta. When the narrative is "on," micro-caps like GRIFFAIN can 2-5x in weeks (cf. the +92% 30d / +40% 7d moves above); when it rotates out, they bleed 80-90%+.
- **Usage vs. speculation.** Unlike pure memes, Griffain claims real product traction (1M+ automated transactions, named agents). If that usage is real and grows, it provides a weak demand floor; if it stalls, the token reverts to meme dynamics.
- **Liquidity-adjusted size.** A ~$16M (or lower) cap with ~$4M daily volume means realistic positions are small; the price is reflexive to a handful of large holders.
