# Hugging Face

**Hugging Face** is the largest open-source AI platform and model hub, founded in 2016 in New York by Clément Delangue, Julien Chaumond, and Thomas Wolf. The Hub passed **1 million hosted models in September 2024** (multiples of that including private repos by 2026), alongside hundreds of thousands of datasets, and its **Transformers** library is the standard Python toolkit for working with [[foundation-models|foundation models]]. For traders, Hugging Face is where you find, download, and deploy financial AI models including [[finbert|FinBERT]], and where open-weight models from [[meta-ai|Meta]] and [[mistral-ai|Mistral]] are distributed.

## Overview

- **Funding/valuation**: $235M Series D in August 2023 at a **$4.5B valuation**, with strategic investors including Google, Amazon, NVIDIA, Salesforce, AMD, Intel, Qualcomm, and IBM (~$400M raised in total across rounds). The company has repeatedly said it is near break-even — unusual among AI startups.
- **2025 expansion into robotics**: acquired French humanoid-robot startup **Pollen Robotics** (April 2025), open-sourcing the **Reachy 2** robot (~$70,000) and backing the **LeRobot** open-source robotics library — positioning HF as the open hub for "physical AI" as well as software models.
- **Inference Providers** (launched 2025): unified API routing to third-party serverless inference (Together, Fireworks, Replicate, etc.), making the Hub a meta-marketplace for model serving.
- Frequently discussed as an IPO candidate in 2026 pre-IPO markets; no confirmed listing as of June 10, 2026.

## Trading Relevance

| Use Case | How Hugging Face Helps |
|----------|----------------------|
| **Sentiment analysis** | Host for [[finbert|FinBERT]] and financial sentiment models |
| **Local model deployment** | Download [[meta-ai|Llama]], [[mistral-ai|Mistral]], Qwen, DeepSeek and other open weights for private trading research |
| **[[fine-tuning-llms|Fine-tuning]]** | Transformers + PEFT libraries for domain-specific model training |
| **[[embeddings-vector-databases|Embeddings]]** | Sentence-transformers for semantic search over financial documents |
| **Model evaluation** | Leaderboards for comparing model quality on financial benchmarks |
| **Dataset sharing** | Financial datasets (earnings calls, SEC filings, price data) |
| **AI-sector signal** | Hub download/trending stats are a free real-time gauge of which model families are winning — useful colour for AI-equity and [[ai-agent-tokens|AI-token]] positioning |

## Key Products

- **Hub**: Model and dataset hosting ("GitHub for AI") — 1M+ public models
- **Transformers**: Python library for using pre-trained models
- **Inference API / Inference Providers**: Run models without managing infrastructure
- **Spaces**: Host ML demos and applications
- **LeRobot + Reachy 2**: Open-source robotics stack (2025)
